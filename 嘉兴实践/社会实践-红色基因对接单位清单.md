## 建议优先级

```
🥇 南湖革命纪念馆 — 首接触，大概率盖章通过
🥇 嘉兴团市委 — 每年接大学生，经验最丰富
🥇 嘉兴关工委 — 口述史素材独特，项目亮点
🥈 区委宣传部 + 党史研究室 — 组合拳，产出深度
🥉 其他 — 备选，看时间和精力
```

## 要确认的 3 件事

1. 是否能盖章（接收单位证明）
2. 是否需要介绍信（学校开具）
3. 实践时间范围（6/25 - 7月）
