# 🗺️ 嘉兴红色实践 · 日间动线地图生成工具

基于高德静态地图 API + PIL，自动为每日日程生成带标注点和动线的地图。

## 原理

```
高德静态地图 API  →  下载基础底图（PNG）
PIL overlay        →  手绘标记（圆形+编号+中文标签）
                    →  手绘动线（彩色线段+方向箭头）
                    →  JPG 输出
```

## 用法

```bash
cd 嘉兴实践/
python3 scripts/gen-itinerary-maps.py
```

输出到 `maps/` 目录，每张图约 85~95 KB。

## 依赖

- `requests` — 拉高德底图
- `Pillow` — 图片合成
- 高德 Web 服务 API Key（自由开发者申请，日均免费额度足）

## 数据来源

- **底图**：高德地图静态图 API（`restapi.amap.com/v3/staticmap`）
- **坐标**：从高德 POI 搜索 API 中提取的南湖区红色资源坐标
- **路线**：基于日程点位顺序绘制的折线（非真实导航路径，仅示意动线）

## 核心技术点

### 坐标 → 像素映射

高德静态地图使用墨卡托投影，给定中心点坐标 + zoom 层级后可估算 1 像素 ≈ 多少米，从而将 WGS84 坐标转为图片像素坐标。

关键公式（嘉兴 30.75°N 附近）：
```python
mpp = 9.5                          # zoom=14 时 1px≈9.5m
LAT_SCALE = 111000                  # 1° 纬度 ≈ 111km
LNG_SCALE = 95300                   # 1° 经度（30.75°N）≈ 95.3km
px = W/2 + (lng-cx) * LNG_SCALE / mpp
py = H/2 - (lat-cy) * LAT_SCALE / mpp   # y 轴翻转
```

### 动线绘制

高德静态 API 免费版不支持返回带 `paths` 参数的路径图（返回 `UNKNOWN_ERROR`），所以用 PIL 手动画在底图上：彩色折线 + 三角箭头标注方向。

### 标注点

圆形标记 + 黑色半透明背景的中文标签，确保在地图任何底色上均可读。

## 已知限制

1. **高德 API 频限**：免费 Key 约 10 QPS 限制，连续请求 4~5 张后触发 `CUQPS_HAS_EXCEEDED_THE_LIMIT`，需 sleep 30s+ 再试。脚本内置了 3s 间隔 + 3 次重试。
2. **坐标精度**：像素映射基于近似公式，zoom 较小时偏差可忽略，zoom≥16 时可能偏移几个像素。
3. **非真实导航**：动线只是点位依次相连的折线，并非高德导航路径规划结果。
