## §3.x 线性回归 & Softmax

### [3.1~3.2] 训练循环四步走
- 困惑：前向 → 算损失 → backward → 更新，每一步在干什么？
- 核心：
  ```
  ① 前向：    y_pred = X@W + b          ← 用当前参数做预测
  ② 算损失：  loss = (y_pred - y)^2      ← 预测和正确答案差多少
  ③ 反向：    loss.backward()            ← 框架自动算每个参数的梯度
  ④ 更新：    param -= lr * param.grad   ← 沿着梯度方向调参数
  ```

### [3.4] Softmax + 交叉熵损失
- 困惑：为什么分类不用平方误差，要用交叉熵？
- 核心：
  ```
  平方误差：关注所有类的分数都接近正确答案 → 对「接近 0 但不够高」的类也要惩罚 → 梯度小，学得慢
  交叉熵：只关注「正确答案的概率」→ log(p) 在 p 接近 0 时梯度巨大 → 学得快
  一句话：分类任务用交叉熵，回归任务用 MSE
  ```

### [3.5] 交叉熵数学
- 困惑：-∑ y·log(p) 这个公式在做什么？
- 核心：y 是 one-hot（正确类=1，其余=0），所以公式简化为 -log(p_正确类)
  ```
  正确类概率 0.9 → -log(0.9) ≈ 0.1  ← 小损失 ✓
  正确类概率 0.1 → -log(0.1) ≈ 2.3  ← 大损失，惩罚狠
  ```

## §4.1 MLP 概念

### [4.1] 为什么要非线性激活函数？
- 困惑：两层 Linear 叠两层 Linear，跟一层有什么区别？
- 核心：
  ```
  没有激活函数：y = (X@W1+b1)@W2+b2 = X@(W1@W2) + (b1@W2+b2) = 一层线性 → 废物
  有 ReLU：     负数砍掉 → 线性变换被打断 → 真正的非线性 → 能学复杂边界
  面试官类比：
    线性 = 只看加权总分
    非线性 = 每个维度独立评判，某个方面不及格直接一票否决
  ```

### [4.2] MLP 参数维度
- 困惑：W1 到底是 (784, 256) 还是 (256, 784)？
- 核心：
  ```
  记住：W 的形状 = (这一层的输入数, 这一层的输出数)
  输入 784 → 隐藏层 256  →  W1 = (784, 256)
  隐藏层 256 → 输出 10   →  W2 = (256, 10)
  验证：X(batch, 784) @ W1(784, 256) → (batch, 256) ✓
  ```

### [4.5] 正则化
- 困惑：这个专业名词忘了是什么意思
- 核心：防止过拟合的「工具包」
  ```
  权重衰减：loss += λ||w||²        ← 权重太大扣分
  Dropout：训练时随机丢神经元      ← 没人可依赖，每个神经元学更扎实
  简化模型：少几层/少几个神经元    ← 不够复杂就没法死记硬背
  加数据：n_train 变大             ← 覆盖更多真实分布
  ```

### [4.6] Dropout 缩放
- 困惑：为什么训练时 dropout 后要除以 (1-p)？
- 核心：
  ```
  训练时随机丢 p% 神经元 → 期望输出 = (1-p) × 全量输出
  测试时不丢 → 输出会变大 1/(1-p) 倍 → 所以训练时提前缩放到期望一致
  不是测试时「加回来」，是训练时提前除好了
  ```

## §6.2 卷积

### [6.2] 卷积核的工作方式
- 困惑（纠正过）：以为卷积是先预处理定位再识别，两阶段
- 核心：
  ```
  错误理解：卷积找区域 → 全连接层识别（两阶段）
  正确理解：卷积+全连接 = 端到端一次训练，所有参数一起反向传播更新
  卷积核不是手设计的，跟全连接层权重一样是学出来的
  ```

## Python 基础

### 这些 import 分别是什么？
- 困惑：`import torch` 那一串分别干什么的
- 核心：
  ```
  import torch                       ← PyTorch 本体，创建张量/矩阵运算
  import torch.nn.functional as F    ← 无参数的函数（relu, softmax, conv2d）
  from torch.utils.data import DataLoader  ← 自动分批+打乱数据
  from torchvision import datasets, transforms  ← 内置数据集+图片预处理
  import matplotlib.pyplot as plt    ← 画图（loss 曲线，权重可视化）
  ```
